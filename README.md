# Sistema de Productos y Presupuestos

Aplicación desarrollada en ASP.NET MVC con base de datos SQLite.

## Funcionalidades
- CRUD de productos
- Creación de presupuestos
- Asociación de productos a presupuestos

## Tecnologías utilizadas

- C#
- ASP.NET MVC
- SQLite
- Entity Framework / ADO.NET (según lo que uses)
- HTML / CSS

## Arquitectura

El proyecto sigue el patrón MVC (Model - View - Controller):

- **Model:** Representa las entidades y la lógica de datos.
- **View:** Interfaces de usuario.
- **Controller:** Maneja las peticiones y la lógica de la aplicación.

También se utiliza el **Repository Pattern** para acceder a los datos.
